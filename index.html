üöÄ How to Set Up

You can easily set it up by following these steps:

  • 1. Install Python 3.8 or later (skip if already installed)
  • 2. Clone or download this repository.
  • 3. Navigate into the downloaded folder and double-click start.bat.
  • 4. Required libraries and dependencies (such as PyTorch and Whisper) will be installed automatically.
  • 5. Once installed, the application will launch automatically.
  • 6. If FFmpeg is not installed, an installation page will open‚Äîplease install it.

üéÆ How to Use

Follow these steps to transcribe your audio files:

  • 1. When the app launches, a file selection window will appear.
  • 2. Select an audio file (.mp3, .wav, etc.).
  • 3. Choose a destination to save the transcribed text.
  • 4. Select the output language from the dropdown menu (e.g., Japanese, English).
  • 5. Choose either CPU or GPU for processing.
  • 6. Select the model to use.
  • 7. Click the "Run" button to start transcription.
  • 8. Once processing is complete, the results will be displayed on the screen.

‚öôÔ∏è CPU & GPU Selection

Whisper GUI Transcription Tool works with CPU, but if a GPU is available, processing will be much faster.

Using GPU

  • If you have a CUDA-capable GPU (e.g., NVIDIA GPU), you can process with GPU.
  • You can select GPU in the options shown after running.
  • To use GPU, CUDA and a compatible version of PyTorch are required (set up automatically by start.bat).

Using CPU

  • Even without a GPU, the tool works perfectly fine with CPU.
  • The setting for CPU is selected automatically by start.bat, so no manual setup is required.
  • Although processing is slower than GPU, the transcription accuracy remains high.

üí° Model Selection

Whisper offers several models, each with its own characteristics:

Available Models:

  • tiny: The lightest and fastest model. Lower accuracy, but extremely quick.
  • base: More accurate than tiny, with moderate speed.
  • small: A balanced model‚Äîgreat combination of accuracy and speed, ideal for general use.
  • medium: High accuracy, slightly slower. Good for complex audio.
  • large: The most accurate model. Slowest, but delivers the highest transcription accuracy.
